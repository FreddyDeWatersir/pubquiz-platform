## PubQuiz Platform Warnars

A real-time pub quiz platform where teams compete by answering multiple-choice questions across multiple rounds, with organizers controlling flow of the quiz.

# Features
- Real-time Updates: Questions appear instantly for all teams when organizers activate new rounds
- Team Management: Simple team registration with persistent sessions
- Multi-round Support
- Organizer Dashboard
- Score Tracking

# Tech Stack
## Backend
- Node.js + Express: RESTful API and server
- Socket.io: WebSocket connections for real-time communication
- SQLite: Lightweight database for data persistence

## Frontend
- React: Interactive user interfaces
- Socket.io-client: real time updates on client side
